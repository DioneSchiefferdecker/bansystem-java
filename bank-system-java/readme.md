# Bank System Java

A simple banking system developed in Java to simulate real-world bank account operations such as deposits, withdrawals, account activation, and balance validation.

## About the Project

This project was created to practice Object-Oriented Programming (OOP) concepts in Java, focusing on clean structure and business rule implementation.

## Features

- Account creation
- Account activation and deactivation
- Deposit operation with validation
- Withdraw operation with balance validation
- Account information display

## Concepts Applied

- Encapsulation
- Constructors
- Method validation
- Business logic implementation
- Object instantiation
- Overriding the toString() method

## Technologies

- Java
